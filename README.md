# AI Ecommerce Product Generator

Developed by Kaif Khan

Final Year Diploma Project

## Overview

An AI-powered ecommerce content generation platform that uses Large Language Models (LLMs) to generate product information from user inputs such as product title, description, and images.

## Features

* Product Title Generation
* Product Description Generation
* Product Summary Generation
* MRP Estimation (Indian Rupees)
* Selling Price Prediction
* Feature Extraction
* Product Category Detection
* Brand Suggestion
* SEO Metadata Generation
* Product Image Analysis
* Key Highlights Generation
* Specifications Generation

## Tech Stack

* React
* Vite
* JavaScript
* OpenRouter API
* Large Language Models (LLMs)
